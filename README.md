# Laboratorio de Análisis Distributivo (LAD)  1.0

El programa de cómputo *Laboratorio de Análisis Distributivo (LAD)* se distribuye mediante repositorios en línea alojados en Google Drive y GitHub bajo dos modalidades:

* Instalador portable.
* Descarga completa de archivos fuente y ejecución local.

---

## 1. Instalador portable

Disponible en:

[Google Drive - Instalador LAD](https://drive.google.com/drive/folders/1sqzEvKEDpPs8XjmE4UlKj5PF5Bem-cy4?usp=drive_link&utm_source=chatgpt.com)

### Procedimiento de instalación

1. Ingrese a la liga anterior.
2. Descargue el archivo `LAD_Instalador.exe` en su computadora.
3. Ejecute el instalador y seleccione la carpeta donde desea extraer los archivos del programa.
4. Una vez concluida la extracción, ejecute el archivo `Iniciar_LAD.bat`.
5. Se abrirá una ventana de ejecución interna del sistema. Esta ventana no debe cerrarse mientras el programa se encuentre en funcionamiento.
6. Posteriormente, la interfaz principal de LAD se abrirá automáticamente en el navegador de Internet predeterminado.

---

## 2. Descarga completa de archivos

Disponible en:

[Google Drive - Archivos completos LAD](https://drive.google.com/drive/folders/1kBMaQ6pHiZRjmXwVovCH3U4WHBNG-VLp?usp=drive_link&utm_source=chatgpt.com)

El desarrollo y actualización del código fuente se realiza directamente mediante GitHub, lo que permite mantener una estructura modular del proyecto y facilitar futuras actualizaciones del sistema.

### Procedimiento de ejecución

1. Ingrese a la liga anterior.
2. Descargue todos los archivos y carpetas en el directorio de su preferencia.
3. Ejecute el archivo `Iniciar_LAD.bat`.
4. Se abrirá una ventana de ejecución interna del sistema. Esta ventana no debe cerrarse mientras el programa se encuentre en funcionamiento.
5. Posteriormente, la interfaz principal de LAD se abrirá automáticamente en el navegador de Internet predeterminado.

---

## Consideraciones generales

En ambas modalidades, el programa fue diseñado para reconocer automáticamente las rutas internas de trabajo dentro de la carpeta de instalación. Por ello, se recomienda conservar sin modificaciones la estructura original de carpetas y archivos distribuida con el sistema.
